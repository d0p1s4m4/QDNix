Building

Prerequisites

  • a C compiler: Clang, GCC
  • Python3
  • GNU Bash
  • GNU Make
  • Git
  • Pandoc (Optional)
  • graphviz (Optional)

Install python dependancies.

$ pip install -r requirements.txt
$ make menuconfig
$ make
